## Sistemas que usa el departamento

| Sistema | Funcion en este departamento |
|---|---|
| **Cloudbeds** | PMS principal: reservas, check-in/out, registro de pagos, inventario de tiendas (Mambo y Playera), estado de habitaciones, llaves electronicas |
| **Odoo** | Registro contable de pagos (Contabilidad → Clientes → Recibo de cliente); recepcion de pedidos de agua, gas y reparaciones |
| **Microsoft Teams** | Carpeta compartida "Ingreso Recepcion": publicacion de pagos y comprobantes; cierre de turno |
| **Visito** | Centralizacion de mensajeria entrante (WhatsApp + Instagram); respuesta automatica por IA; modo manual para atencion humana |
| **TT Hotel** | Programacion de cerraduras inteligentes; modulos 4, 2, 7-1, 7-2; requiere usuario recepcion para llaves de huesped, usuario admin para llaves maestras |
| **Time Locks / Lockia** | Programacion de cerraduras magneticas (habitaciones en transicion); campos: numero habitacion + numero noches + cantidad tarjetas |
| **Poster** | POS de A&B; sincroniza comandas automaticamente a Cloudbeds; recepcion no accede directamente |
| **Excel** | Formato de cierre de turno (registro manual de pagos por metodo) |
| **WhatsApp** | Comunicacion interna: recepcion ↔ administracion (validacion de pagos), Adriana ↔ equipo, ama de llaves → recepcion (estado habitaciones) |
| **BCV (Banco Central de Venezuela)** | Consulta de tasa euro/bolivar para calcular equivalencias en check-in/out |
